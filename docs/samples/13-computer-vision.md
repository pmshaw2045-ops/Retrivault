# 计算机视觉

## 图像分类

图像分类是 CV 的基础任务，目标是给输入图像分配类别标签。

### 经典架构

- **AlexNet（2012）**：深度学习在 CV 领域的突破，大幅超越传统方法。
- **ResNet（2015）**：引入残差连接，解决深层网络退化问题，可训练 152 层网络。
- **EfficientNet（2019）**：使用神经架构搜索同时优化深度、宽度和分辨率。

## 目标检测

目标检测不仅识别物体类别，还要定位其在图像中的位置。

- **YOLO（2016）**：将检测视为回归问题，单次前向传播即可完成，速度极快。
- **Faster R-CNN（2015）**：两阶段方法，先推荐区域再分类，精度高但较慢。
- **DETR（2020）**：基于 Transformer 的端到端检测，无需手工设计的锚框。

## 图像分割

- **语义分割**：对每个像素分类（如道路、行人、车辆）。
- **实例分割**：不仅区分类别，还区分同一类别的不同实例。
- **Mask R-CNN**：Faster R-CNN 的扩展，同时输出检测框和分割掩码。

## 图像生成

- **GAN（2014）**：生成对抗网络，由生成器和判别器对抗训练。
- **扩散模型（2020）**：通过逐步去噪生成图像，Stable Diffusion 是其代表。
- **ControlNet**：对扩散模型添加条件控制，实现精确姿态/边缘引导生成。
