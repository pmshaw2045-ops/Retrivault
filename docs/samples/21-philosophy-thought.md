# 哲学思想流派

## 西方哲学

### 古希腊哲学

- **苏格拉底**：辩证法（诘问法），"我知道我一无所知"。
- **柏拉图**：理念论，现实世界是理念世界的影子。《理想国》探讨正义和理想城邦。
- **亚里士多德**：形而上学、逻辑学、伦理学。四因说（质料因、形式因、动力因、目的因）。

### 近代哲学

- **笛卡尔**："我思故我在"，理性主义，心物二元论。
- **康德**：物自体与现象界，"人为自然立法"。三大批判（纯粹理性、实践理性、判断力）。
- **尼采**：超人哲学、权力意志、"上帝已死"、永恒轮回。

### 现代哲学

- **存在主义（萨特）**："存在先于本质"，人有选择的自由和责任。
- **分析哲学（维特根斯坦）**：语言的界限就是世界的界限，哲学问题是语言问题。

## 东方哲学

### 儒家

孔子核心思想：仁（爱人）、礼（社会规范）、中庸（不偏不倚）。孟子发展性善论，荀子主张性恶论。

### 道家

老子《道德经》："道可道，非常道"。"无为"不是不作为，而是顺应自然规律。庄子齐物论，万物平等。

### 佛教哲学

四圣谛：苦集灭道。缘起性空（一切事物因缘和合，无独立自性）。禅宗强调"不立文字，直指本心"。
