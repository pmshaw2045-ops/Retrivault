# 经济学原理

## 供需理论

### 需求

需求定律：其他条件不变时，价格上升，需求量下降。需求曲线向右下倾斜。

影响需求的因素：收入、偏好、替代品价格、互补品价格、预期。

### 供给

供给定律：价格上升，供给量上升。供给曲线向右上倾斜。

影响供给的因素：生产成本、技术、税收/补贴、预期。

### 市场均衡

当需求等于供给时达到均衡价格和数量。供给过剩（价格过高）或短缺（价格过低）会通过价格机制自动调节。

## 市场结构

| 类型 | 厂商数量 | 产品差异 | 进入壁垒 | 例子 |
|------|----------|----------|----------|------|
| 完全竞争 | 极多 | 无 | 无 | 农产品 |
| 垄断竞争 | 多 | 有差异 | 低 | 餐馆、服装 |
| 寡头垄断 | 少数 | 可有 | 高 | 电信、航空 |
| 完全垄断 | 一个 | 无替代品 | 极高 | 公用事业 |

## 宏观经济指标

- **GDP**：国内生产总值，衡量经济规模。
- **CPI**：消费者价格指数，衡量通货膨胀。
- **失业率**：失业人口占劳动力的比例。
- **利率**：央行通过调整基准利率影响信贷和通胀。

## 财政与货币政策

- **财政政策**：政府通过税收和支出调节经济（凯恩斯主义）。
- **货币政策**：央行通过利率和货币供给调节经济（量化宽松、加息）。
