# 数学基础

## 微积分

### 导数

导数描述函数在某点的变化率。$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

常见导数：$(x^n)' = nx^{n-1}$，$(\sin x)' = \cos x$，$(e^x)' = e^x$

### 积分

积分是导数的逆运算，计算曲线下的面积。$$\int_a^b f(x) \, dx = F(b) - F(a)$$

## 线性代数

### 向量与矩阵

向量是有大小和方向的量。矩阵是二维数组，表示线性变换。

- **矩阵乘法**：\(C_{ij} = \sum_k A_{ik} B_{kj}\)
- **特征值/特征向量**：\(A\mathbf{v} = \lambda \mathbf{v}\)
- **奇异值分解（SVD）**：\(A = U\Sigma V^T\)

### 应用

线性代数在机器学习中无处不在——神经网络的前向传播就是矩阵乘法链。主成分分析（PCA）基于特征值分解。推荐系统的协同过滤基于矩阵分解。

## 概率论

### 基本概念

- **条件概率**：\(P(A|B) = P(A \cap B) / P(B)\)
- **贝叶斯定理**：\(P(A|B) = P(B|A)P(A) / P(B)\)
- **期望与方差**：$E[X] = \sum x \cdot P(x)$，$Var[X] = E[X^2] - E[X]^2$

### 重要分布

- **正态分布**：$N(\mu, \sigma^2)$，中心极限定理的基础
- **二项分布**：$B(n,p)$，n 次伯努利实验的成功次数
