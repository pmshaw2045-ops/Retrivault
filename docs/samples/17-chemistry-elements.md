# 元素化学

## 元素周期表

元素周期表按原子序数排列，同一族的元素具有相似的化学性质。主族包括：

- **碱金属（IA）**：Li、Na、K 等，最外层 1 个电子，活泼金属。
- **卤素（VIIA）**：F、Cl、Br、I，最外层 7 个电子，强氧化性。
- **稀有气体（VIIIA）**：He、Ne、Ar 等，最外层 8 个电子，化学惰性。

## 化学键

### 离子键

金属和非金属之间通过电子转移形成。NaCl（氯化钠）是典型例子：Na 失去电子变成 Na⁺，Cl 得到电子变成 Cl⁻。

### 共价键

非金属之间通过共享电子对形成。H₂O 中 O 和 H 共享电子，形成极性共价键。CO₂ 中的 C=O 双键是非极性共价键。

### 金属键

金属原子之间通过自由电子形成"电子海"结构，赋予金属良好的导电性和延展性。

## 化学反应类型

- **氧化还原**：电子转移，如燃烧、腐蚀、置换反应。
- **酸碱中和**：H⁺ + OH⁻ → H₂O，包括强酸强碱和弱酸弱碱。
- **沉淀反应**：生成不溶性盐，如 AgNO₃ + NaCl → AgCl↓ + NaNO₃。

## 热力学基础

- **焓（ΔH）**：反应热，放热反应 ΔH < 0。
- **熵（ΔS）**：系统混乱度，自发反应倾向于 ΔS 增加。
- **吉布斯自由能（ΔG = ΔH - TΔS）**：判断反应自发性，ΔG < 0 自发。
