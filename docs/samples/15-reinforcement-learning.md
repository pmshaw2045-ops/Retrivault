# 强化学习

## 核心概念

强化学习（RL）研究智能体（Agent）如何在环境中通过试错学习最优策略。

- **状态（State, s）**：环境的当前描述。
- **动作（Action, a）**：智能体可以执行的操作。
- **奖励（Reward, r）**：环境反馈的标量信号。
- **策略（Policy, π）**：从状态到动作的映射函数。
- **价值函数（Value Function, V/Q）**：评估状态或状态-动作对的长期回报。

## 主要算法

### 基于价值

- **Q-Learning**：通过更新 Q 表学习最优动作价值函数。
- **DQN（2013）**：用深度神经网络近似 Q 函数，引入经验回放和目标网络。
- **Double DQN**：解决 DQN 过估计问题，用两个网络解耦选择和评估。

### 基于策略

- **Policy Gradient（REINFORCE）**：直接优化策略参数，适合连续动作空间。
- **PPO（2017）**：裁剪的代理目标函数，训练稳定，OpenAI 默认算法。
- **SAC（2018）**：软演员-评论家，最大化期望回报+熵，探索性好。

### 基于模型

- **AlphaGo（2016）**：蒙特卡洛树搜索 + 深度神经网络，击败人类围棋冠军。
- **MuZero（2019）**：无需环境模型，通过隐式学习环境动态进行规划。

## 应用场景

- 游戏（围棋、Dota 2、星际争霸）
- 机器人控制（抓取、行走）
- 推荐系统（用户交互序列优化）
- 自动驾驶（决策规划层）
