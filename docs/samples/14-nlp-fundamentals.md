# 自然语言处理基础

## 语言表示方法

### 词嵌入（Word Embeddings）

将词汇映射到低维稠密向量空间，语义相近的词在向量空间中距离相近。

- **Word2Vec（2013）**：CBOW 和 Skip-gram 两种架构，通过上下文预测词。
- **GloVe（2014）**：利用全局词共现统计，结合矩阵分解。
- **FastText（2016）**：考虑子词信息，能处理未登录词。

### 上下文表示

- **ELMo（2018）**：双向 LSTM 生成上下文化的词表示。
- **BERT（2018）**：基于 Transformer 的双向编码器，通过 Masked LM 预训练。
- **GPT 系列**：单向自回归语言模型，通过下一个词预测预训练。

## 核心任务

| 任务 | 输入 | 输出 | 典型模型 |
|------|------|------|----------|
| 文本分类 | 文本 | 类别 | BERT, TextCNN |
| 命名实体识别 | 文本 | 实体+类型 | BERT-CRF |
| 关系抽取 | 文本 | 实体对+关系 | SpanBERT |
| 文本摘要 | 长文本 | 摘要 | BART, Pegasus |
| 机器翻译 | 源语言 | 目标语言 | Transformer |

## 评价指标

- **BLEU**：机器翻译，基于 n-gram 精确率匹配。
- **ROUGE**：文本摘要，基于召回率的 n-gram 匹配。
- **Perplexity**：语言模型的困惑度，越低越好。
